Here is the full list of changes:

  • Fast server browser
  • Next/Prev weapon keys
  • Lean forward key
  • Cooperative gametype added, with a bunch of options
  • Balanced teams forced on in Thieves vs Guards gametype
  • Shorter pre-game wait
  • Shorter post-game wait
  • Loadout hotkey confirmation
  • Can no longer use loadout buy keys midgame
  • Check objectives on player ZoneChange, so thieves don't have to wait for a random amount of time in exit zone for objectives to kick in
  • AI guards correctly follow player's footprints
  • AI guards can relight torches
  • AI guards can use the supply chest
  • Guards with readied firebolts have glowing crossbows in 3rd person view (handy warning)
  • Support for thievery:// links in webpages or in the Thievery chat
  • Password joining fixed
  • Can no longer do speech taunts while dead
  • Forced delay between each taunt
  • Rats can no longer taunt
  • Spectator's scoreboard and objectives no longer clash in pregame if you have them both up
  • Rats can now hunt down and kill other rats for bonus points, killer rat scores shown on the F2 screen
  • Can now use the 'Test Map' button in Thievery UnrealEd
  • Rats not longer make loud splashing noises as they go in and out of water
  • Movers correctly block light on dedicated servers
  • Guards no longer drop the supply chest at the thief spawn if they disconnect in the pregame stage
  • If you join a server when game has ended, you'll go into the end scores
  • Fixed some TestCanSeeMe crashes
  • Fixed the nextmap not showing up at the end sometimes
  • Rope cylinder is placed around rope, not hit location of arrow
  • NextMap correctly set in co-op/thiefmatch games
  • Altered ladder stuff slightly to be better when two ladders overlap
  • Force to spectate if you reconnect after being turned into a rat
  • Rats have a short range biting attack that can only hurt other rats
  • Rat death messages only shown to other rats
  • Spectators can no longer get blinded
  • Hear sound volumes correctly when in chasecam
  • Scoreboard should only show teams 0 and 1 (not team 2 bots)
  • Disabled the web admin by default
  • Change the max client rate and number of players by default
  • Turn off directsound by default
  • Accept downloads off by default
  • Added Paz and Vigil to the credits
  • Offline mapping and play guide docs included
